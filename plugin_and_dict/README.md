OmegaT Plugins and Dictionaries
===============================

ここには、OmegaTで使うプラグインと辞書のアーカイブを置いています。

辞書は、すでにプロジェクトのdictionaryフォルダに配置してあります。
pyglossaryは、FreeDictをOmegaTで利用するためにStarDict形式へ変換するためのツールで翻訳作業には必要ありません。

プラグインは、アーカイブを展開して、それぞれの中にあるjarファイルをOmegaT本体のpluginsフォルダに配置してください。
